## ðŸªœ Step-by-Step Summary

1. Open EC2 Console
I navigated to EC2 from the AWS Services menu.

### 2. Launch Instance
Clicked **Launch instance** and named my instance.

### 3. Choose AMI
Selected **Amazon Linux 2023** (Free Tier eligible).

### 4. Choose Instance Type
Selected **t2.micro** (Free Tier eligible).

### 5. Create Key Pair
Created a new key pair to securely connect to the instance.

### 6. Configure Network
Enabled auto-assign public IP and created a security group allowing SSH from my IP.

### 7. Launch
Clicked **Launch instance** and waited for the instance to enter the **running** state.
